• Project Description : 
This project focuses on the development of Quix, a quiz and study tracker application used to improve learning effectiveness and support skill development among students. The application enables students to take quizzes, monitor their time studying and track their study progress, motivating them to actively learn and manage their time better.

• Overview of Your SDLC Work :
Planning – An online meeting was conducted to discuss the SDLC processes, understand the requirements, decide on the target users, and measure the feasibility of the project.
Design –The design development followed an iterative process from low-fidelity wireframes completed in Sprint 1 to high-fidelity mockups in Sprint 2. 
Testing – Internal testing was done by group members in Figma to identify issues, and the prototype was reviewed by external peers for User Acceptance Testing (UAT).
